What is binary?

Computers are not very clever things. All they can understand are '1' and '0' which you would think would be very limiting, right?

Wrong. Every single thing you can see on your screen right now is made up of these 1s and 0s.

Your computer (and everything else electronic!) converts everything to this language so that it can understand what it needs to do.

We call this language 'binary' and we're about to learn to convert pretty much anything to it.

How does binary work?

To understand binary you need to understand how powers work. You'll be familiar with them even if you don't know that they are powers! The decimal system, the number system you are familiar, is base 10. Each column is ten times bigger than the one before it.

Binary is base two - each column to two times bigger than the one before it.

Can you remember learning how to represent numbers in primary school? You would have a units column, a tens column, a hundreds column and so on.

With the decimal system, a 1 in the tens column represents 1 ten. A 2 in the tens column represents 2 tens. This can increase all the way up to 9 in each column.

Binary uses the same concept, except the only options in each column are 1 and 0. A 1 in the one column represents 1 one. A 1 in the two column represents two. A 1 in both a one and two column equals three as you just add the columns together.
